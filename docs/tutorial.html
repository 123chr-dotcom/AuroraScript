AuroraScript 使用教程

基本使用指南

1. 启动 AuroraScript 编辑器

# 通过命令行启动
$ AuroraScript [文件名]

2. 创建新文件

文件 → 新建 或 Ctrl+N

3. 保存文件

文件 → 保存 或 Ctrl+S

编辑功能

1. 多光标编辑

按住Alt键点击可以添加多个光标，实现同时编辑多处内容。

2. 代码折叠

点击行号旁边的折叠图标可以折叠代码块。

3. 查找替换

编辑 → 查找 或 Ctrl+F
编辑 → 替换 或 Ctrl+H

高级功能

1. 代码片段

使用Tab键触发预定义的代码片段。

2. 命令行工具

# 批量处理文件
$ AuroraScript --batch *.txt

3. 插件系统

工具 → 插件管理器

常见问题

Q: 如何更改主题？
A: 通过设置 → 外观 → 主题选择器

Q: 编辑器启动慢怎么办？
A: 尝试禁用不必要的插件，或重置设置。

Q: 如何报告问题？
A: 通过帮助 → 反馈问题提交报告。
