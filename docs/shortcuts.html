AuroraScript 快捷键指南

常用快捷键

功能 快捷键 说明
新建文件 Ctrl+N 创建一个新文件
打开文件 Ctrl+O 打开现有文件
保存文件 Ctrl+S 保存当前文件
查找 Ctrl+F 在当前文件中查找文本
替换 Ctrl+H 查找并替换文本
复制行 Ctrl+D 复制当前行或选中内容
删除行 Ctrl+Shift+K 删除当前行
注释行 Ctrl+/ 注释或取消注释当前行

编辑快捷键

功能 快捷键 说明
多光标选择 Alt+Click 添加多个光标
选择所有匹配项 Ctrl+Shift+L 选择当前选中内容的所有匹配项
格式化代码 Shift+Alt+F 格式化当前文件或选中内容
跳转到行 Ctrl+G 跳转到指定行号

自定义快捷键

您可以通过以下步骤自定义快捷键：

  1. 打开设置菜单 (文件 → 设置)
  2. 选择"键盘快捷键"选项卡
  3. 搜索要修改的命令
  4. 点击命令旁边的快捷键区域
  5. 按下新的快捷键组合
  6. 点击"保存"按钮应用更改
